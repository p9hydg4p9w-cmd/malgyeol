# 말결 3.0

온라인 비하 표현과 조롱 문화에서 파생된 말투를 문맥에 따라 점검하고,
더 존중하는 표현을 제안하는 발표용 프로토타입입니다.

## 핵심 기능
- `개웃기노 이기` 탐지 오류 수정
- 인터넷식 `-노` 종결 표현 탐지
- 실제 방언 가능성 별도 안내
- 추임새 `이기` 탐지
- 호칭형 `게이야`, `게이들아` 탐지
- 직접 비하어와 집단 일반화 탐지
- 말의 온도 점수
- 표현별 색상 강조
- 발견 표현별 이유와 맥락·유래 설명
- 추천 표현 3개
- 접었다 펼치는 교육 설명

## GitHub 적용
기존 저장소에서 다음 파일을 새 파일로 교체합니다.

- index.html
- style.css
- script.js
- README.md

업로드 후 Commit changes를 누르면 GitHub Pages가 자동으로 다시 배포됩니다.
