# 말결 2.0 — 맥락 판단 버전

## 추가된 기능
- 인터넷식 `-노` 종결 표현 탐지
- 경상도 방언 가능성 별도 안내
- 추임새 `이기` 탐지
- 호칭형 `게이야`, `게이들아` 탐지
- 정체성을 설명하는 정상 문장인 `그는 게이다`는 탐지하지 않음
- `-노 + 이기` 결합 시 커뮤니티식 말투 가능성 추가 안내
- 직접 비하 / 조롱 문화 파생 표현 / 문맥 확인 표현을 서로 다른 단계로 분류

## GitHub에 적용하는 방법
기존 저장소에서 index.html, style.css, script.js를 이 파일들로 교체한 뒤 Commit changes를 누르면 됩니다.
README.md도 함께 교체할 수 있습니다.

## 주의
이 도구는 교육용 규칙 기반 프로그램입니다.
방언과 실제 발화 맥락을 완벽히 판단할 수 없으므로 사람이나 문장을 단정하는 용도로 사용하지 않습니다.
