# 말결 5.0

말결 5.0은 표현을 다음 유형으로 나누어 분석합니다.

- 직접 비하
- 집단 일반화
- 조롱 문화에서 파생된 표현
- 문맥 확인
- 비격식 인터넷 은어
- 거친 표현

## 주요 추가 기능
- 1차 선별 표현 DB 확대
- 말의 온도
- 존중성 / 공격성 / 일반화 / 은어성 / 공식성 지표
- `개추`, `비추`, `억까`, `억빠`, `뇌절`, `팩폭` 등 비격식 표현 분류
- 직접 혐오표현과 비격식 표현을 서로 다르게 안내
- 문장 패턴 탐지

## 적용 방법
기존 GitHub 저장소에서 아래 파일을 교체합니다.

- index.html
- style.css
- script.js
- README.md

그 뒤 Commit changes를 누르면 됩니다.
