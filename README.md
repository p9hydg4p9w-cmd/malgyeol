## 6.1 추가 기능
- 새 표현을 사용자가 직접 등록
- 등록 직후 분석과 표현 사전에 반영
- 브라우저 localStorage에 보관
- 개별 삭제와 전체 삭제
- 추가 표현 목록을 JSON 파일로 내보내기

GitHub Pages는 서버가 없는 정적 사이트이므로 여러 사람의 제보가 운영자에게 자동 전송되지는 않습니다. 현재 버전은 각 기기에 저장하고 JSON으로 내보내도록 구성했습니다.
